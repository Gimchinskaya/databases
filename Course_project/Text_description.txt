Общее текстовое описание БД и решаемых ею задач

В данном проекте реализована база данных booking, созданная на основе сайта https://www.booking.com
В БД есть следующие таблицы:
	users, profiles - таблицы с данными пользователей,
	stays, stays_types - таблицы c основными данными жилья и его типом,
	booking - таблица бронирования жилья
	media_stays, media_types, photo_profiles - таблицы медиафайлов жилья и пользователей,
	main_settings, facilities, food, distance_type_of_beach, distance_type_of_center - таблицы параметров жилья и типов этих параметров,
	reviews_and_rating - таблица отзывов и рейтингов жилья.
	
Данная БД позволяет решать следующие задачи:
	1) поиск жилья по определенным параметрам (цена, город, расположение относительно центра/пляжа, рейтинг пользователей, количество звезд, тип питания, наличие различных удобств),
	2) поиск свободного жилья в городе на выбранные даты,
	3) бронирование жилья,
	4) просмотр статистики бронирования определенного жилья,
	5) просмотр статистики бронирования по пользователям,
	6) просмотр всех или только интересующих параметров жилья (см. пункт 1),
	7) просмотр отзывов и рейтингов пользователя о жилье.
